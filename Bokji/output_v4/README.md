# Bokji output_v4 — 산출물 설명

> **분석 주제**: 서울시 노인 보행일상권 ② 복지·녹지 인프라 접근성 분석  
> **기준 시간**: 도보 30분  
> **적용 모델**: OSM 보행 네트워크 Dijkstra + Tobler 경사 보정

---

## 변경 이력 (v3 → v4)

| 항목 | v3 | v4 |
|------|----|----|
| 핵심 지표 | 취약도·박탈 노인 수 | **도달가능점수** (선택속도 도달 수 / 일반인 도달 수 × 100) |
| 경사 보정 | 미적용 | **Tobler 보행속도 모델** — 동별 OSM 경사도 기반 보정 |
| 속도 계층 | 일반인·노인·보조기기·하위15% | 동일 + **경사보정 3종** (노인·보조기기·하위15%) |
| 집계 단위 | 동(洞) 단위만 | 동 단위 + **구(區) 단위 도달가능점수** |
| 대시보드 | 구별 반경 원만 | + **동별 반경 원** 토글, 경사로 보정 온/오프 |
| 결론 페이지 | 없음 | **conclusion.html** — 경사 보정 전후 비교 종합 |

---

## 생성 스크립트 실행 순서

```
# 1. Tobler 보정 Dijkstra 재계산 (약 15분 소요)
python compute_tobler_v4.py

# 2. 구 단위 점수 집계 CSV 추출
python extract_gu_scores_v4.py

# 3. 대시보드 HTML 생성
python generate_dashboard_v4.py

# 4. 결론 HTML 생성
python generate_conclusion_v4.py
```

### 1. `dong_reachability_v4.csv`

**크기**: 45.5 KB · **행**: 426 (서울시 전체 행정동) · **열**: 25

동(洞) 단위 원본 데이터. 기존 v2 결과에 Tobler 비율과 경사 보정 도달 수를 추가한 마스터 테이블이다.

#### 컬럼 구성

| 컬럼 | 설명 |
|------|------|
| `동_key` | `구명_동명` 형태의 고유 키 (예: `종로구_사직동`) |
| `구명` / `동명` | 자치구명·행정동명 |
| `65세이상인구` / `고령화율` | 동별 고령 인구 현황 |
| `복지_일반인` | 일반인 속도(1.39 m/s) 기준 30분 내 복지시설 수 |
| `복지_일반노인` | 노인 속도(1.12 m/s) 기준 복지시설 수 |
| `복지_보조기기` | 보조기기 속도(0.88 m/s) 기준 복지시설 수 |
| `복지_보조하위15p` | 하위15% 속도(0.70 m/s) 기준 복지시설 수 |
| `공원_*` | 위와 동일, 공원 기준 |
| `복지_속도격차` / `공원_속도격차` | 일반인 대비 하위15% 도달 수 차이 |
| `복지_박탈노인` / `공원_박탈노인` | 속도격차 × 65세이상인구 (v2 취약도 지표) |
| `vulnerability_v2` | 구 내 상대 취약도 순위 (v2 레거시) |
| **`tobler_ratio`** | 동별 Tobler 경사 보정 계수 (범위: 0.320 ~ 0.989, 평균: 0.801) |
| **`복지_일반노인보정`** | `tobler_ratio` 적용 후 노인 속도 기준 복지시설 수 |
| **`복지_보조기기보정`** | `tobler_ratio` 적용 후 보조기기 속도 기준 복지시설 수 |
| **`복지_보조하위15p보정`** | `tobler_ratio` 적용 후 하위15% 속도 기준 복지시설 수 |
| **`공원_일반노인보정`** / **`공원_보조기기보정`** / **`공원_보조하위15p보정`** | 위와 동일, 공원 기준 |

#### 주요 통계

| 항목 | 값 |
|------|----|
| 경사 가장 급한 동 | 노원구 중계4동 (tobler_ratio = 0.320) |
| 경사 가장 완만한 동 | 영등포구 당산1동 (tobler_ratio = 0.989) |
| 복지_일반인 = 0 (분모 결측) | 37개 동 |
| 공원_일반인 = 0 (분모 결측) | 74개 동 |

> **도달가능점수 분모가 0인 동은 구 평균 계산에서 제외**된다.  
> 일반인 속도는 Tobler 보정 대상에서 제외 — 경사 무관 기준선으로 고정 사용.

### 2. `gu_scores_v4.csv`

**크기**: 1.2 KB · **행**: 25 (서울시 25개 자치구) · **열**: 7

구(區) 단위 도달가능점수 집계 테이블. 경사 보정 전후 3개 속도 유형 점수를 모두 포함한다.

#### 컬럼 구성

| 컬럼 | 설명 |
|------|------|
| `구명` | 자치구명 |
| `점수_노인_경사X` | 일반 노인 속도(1.12 m/s) 기준 도달가능점수, 경사 보정 없음 |
| `점수_노인_경사O` | 일반 노인 속도 × Tobler 보정 후 도달가능점수 |
| `점수_보행보조_경사X` | 보조기기 속도(0.88 m/s) 기준, 경사 보정 없음 |
| `점수_보행보조_경사O` | 보조기기 속도 × Tobler 보정 후 |
| `점수_하위15_경사X` | 하위15% 속도(0.70 m/s) 기준, 경사 보정 없음 |
| `점수_하위15_경사O` | 하위15% 속도 × Tobler 보정 후 |

#### 도달가능점수 산식

```
도달가능점수 = (해당 속도로 도달 가능한 복지 + 공원 수) / (일반인 도달 복지 + 공원 수) × 100
```

구 단위 점수 = 구 내 행정동 평균 (분모 = 0인 동 제외)

#### 서울시 전체 평균 (25개 구 평균)

| 유형 | 경사X | 경사O | 감소폭 |
|------|-------|-------|--------|
| 일반 노인 | 76.2점 | 49.5점 | −26.7점 |
| 보조기기 | 45.4점 | 29.8점 | −15.6점 |
| 하위15% | 28.0점 | 18.3점 | −9.7점 |

#### 경사 보정 후 최저·최고 구 (하위15% 경사O 기준)

| 순위 | 구 | 점수 |
|------|-----|------|
| 최저 | 성북구 | 8.1점 |
| 최저 2위 | 서초구 | 8.6점 |
| 최저 3위 | 관악구 | 8.9점 |
| 최고 | 광진구 | 35.0점 |

### 3. `infra_dashboard_bokji.html`

**크기**: 153.7 KB · **독립 실행형 HTML** (외부 CDN: Leaflet, Chart.js)

서울시 복지·녹지 인프라 접근성 인터랙티브 대시보드.

#### 주요 기능

| 기능 | 설명 |
|------|------|
| **구 선택** | 25개 자치구 드롭다운 — 선택 시 지도·차트·테이블 동기화 |
| **보행 속도 선택** | 일반인 / 일반 노인 / 보조기기 / 하위15% 4단계 |
| **시설 유형 필터** | 복지시설 / 공원 / 전체 |
| **지도 레이어** | 복지시설·공원 마커, 구 경계 반경 원 |
| **경사로 보정 토글** | ON 시 Tobler 보정 속도로 도달가능점수·반경 재계산 (일반인 제외) |
| **동별 반경 표시 토글** | 선택 구 내 모든 행정동의 도달 반경 원을 점수 색상으로 표시; 50점 미만 동에 레이블 |
| **도달가능점수 패널** | 복지·공원 개별 점수 + 합산 점수 (일반인 선택 시 숨김) |
| **TOP10 차트** | 일반인: 취약도 순위 / 노인·보조·하위15%: 도달가능점수 하위 순 |
| **동별 테이블** | 선택 속도 기준 도달 수·취약도·점수 정렬 |

#### 데이터 구조 (내장 JSON)

```

             w[4], p[4],          ← 경사보정 전 도달 수 (속도 0~3)

             tobler,              ← 동별 Tobler 비율
             clat, clng }         ← 행정동 중심점 좌표
WELFARE[189]: { name, addr, lat, lng }
PARK[132]:    { name, area, lat, lng }
```

### 4. `conclusion.html`

**크기**: 17.9 KB · **독립 실행형 HTML** (외부 CDN: Chart.js)

구 단위 도달가능점수를 기반으로 경사 보정 전후를 비교하는 종합 결론 대시보드.

#### 구성 섹션

| 섹션 | 내용 |
|------|------|
| **요약 카드** | 서울시 25개 구 평균 점수를 3개 유형별로 경사X → 경사O 표시; 70점 이상 녹색·50점 이상 주황·미만 빨강 |
| **핵심 발견** | 하위15% 경사O 최저·최고 구, 경사 보정으로 최대 점수 감소 구 |
| **비교 차트** | 유형 탭 전환 (노인·보조기기·하위15%) × 경사X/경사O 그룹 막대 차트 (경사O 오름차순 정렬) |
| **감소량 차트** | 보조기기 vs 하위15% 감소량 나란히, 3개 유형 동시 비교 (하위15% 내림차순) |
| **종합 테이블** | 25개 구 × 9개 수치 (경사X·경사O·감소량 × 3개 유형) |

#### 핵심 발견 (경사보정 후 최악 상위 3구 — 하위15%)

| 구 | 경사X | 경사O | 감소폭 |
|----|-------|-------|--------|
| 성북구 | 27.9 | **8.1** | −19.8점 |
| 서초구 | 20.3 | **8.6** | −11.7점 |
| 관악구 | 18.1 | **8.9** | −9.2점 |

> 은평구는 노인 기준 경사 보정 감소가 −45.3점으로 서울 최대.  
> 서대문구는 보조기기 기준 경사 보정 감소가 −28.2점으로 서울 최대.

## 주요 개념 정의

| 용어 | 정의 |
|------|------|
| **Tobler 보행속도 모델** | 경사도에 따른 보행 속도 보정 계수. 평탄지 = 1.0, 경사 클수록 감소 |
| **도달가능점수** | `(선택 속도 도달 수 / 일반인 도달 수) × 100`. 일반인 기준 100, 낮을수록 취약 |
| **경사X / 경사O** | 경사 보정 미적용 / 적용. 일반인 속도는 경사O도 보정 없음 (기준선) |
| **하위15%** | 보조기기 사용자 중 보행 속도 하위 15% (0.70 m/s) |
| **N/A (분모=0)** | 일반인 기준 도달 시설이 0개인 동 — 점수 산출 불가, 구 평균 제외 |
