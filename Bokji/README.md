# 서울시 노인 생활복지 및 녹지 접근성 분석

서울시 어르신들이 **도보로 복지시설·공원에 접근할 수 있는지**를 분석하고,  
청년과 노인의 보행 속도 차이에 따른 **접근성 격차**와 **복지 사각지대**를 시각화한 파이프라인입니다.

---

## 디렉토리 구조

```
Bokji/
├── analysis.py                  ← 메인 분석 파이프라인 (Step 1~6)
├── visualize_welfare.py         ← 복지시설 단독 시각화
├── visualize_parks.py           ← 공원·녹지 단독 시각화
├── visualize_dong.py            ← 행정동 단위 정밀 분석
│
├── 서울시 사회복지시설(노인여가복지시설) 목록.csv   ← 입력 데이터
├── 서울시 주요 공원현황(2026 상반기).xlsx           ← 입력 데이터
├── 고령자현황_20260421103806.csv                    ← 입력 데이터
│
└── output/                      ← 분석 결과물 (실행 후 생성)
    ├── 지도 (HTML)
    ├── 차트 (PNG)
    └── 데이터 (CSV)
```

## 핵심 분석 개념

### 보행 속도 기준

노인과 청년의 실제 보행 속도 차이를 반영합니다.

| 구분 | 속도 | 10분 거리 | 20분 거리 |
|------|------|-----------|-----------|
| 노인 | 2.4 km/h | **400m** | 800m |
| 청년 | 4.8 km/h | **800m** | 1,600m |

→ 같은 시설이라도 청년에게는 보행권 내에 있고 노인에게는 사각지대인 경우가 발생합니다.

### 버퍼 분석 (Buffer Analysis)

각 시설·공원을 중심으로 반경 원을 그려 보행권을 계산합니다.

```
노인 도보 10분 = 400m 반경 원
청년 도보 10분 = 800m 반경 원
```

### 평가 지표

| 지표 | 공식 | 의미 |
|------|------|------|
| **Welfare Index** | 시설 수 / 65세이상인구 × 10,000 | 1만명당 시설 수 |
| **Green Index** | 공원 면적(㎡) / 65세이상인구 | 1인당 공원 면적 |
| **보행권 커버리지** | 버퍼 합집합 면적 / 자치구(동) 면적 × 100 | 보행권 안에 드는 지역 비율 (%) |
| **속도 격차** | 청년 커버리지 − 노인 커버리지 | 노인이 추가로 소외되는 비율 (%p) |
| **박탈 노인 수** | 65세이상인구 × (1 − 노인 커버리지 / 100) | 보행권 밖 추정 노인 수 (명) |

### Vulnerability Score (복지 결핍 지수)

노인 기준으로 보행권에서 박탈된 인구 규모를 종합한 지수입니다.

```
결핍 지수 = 복지 박탈 노인 수(정규화) × 0.50
          + 공원 박탈 노인 수(정규화) × 0.50
```

> 고령화율은 제외했습니다. 고령화율과 접근성 박탈 간 상관관계가 유의미하지 않아 (r=0.020, p=0.68),  
> 인구 규모 기반의 직접 지표를 사용합니다.

## 분석 결과 해석 가이드

### 지도에서 확인할 것

- **붉은 색(YlOrRd)이 진할수록** 결핍 지수가 높은 취약 지역
- **노인 버퍼(빨간 원)** 바깥에 있는 지역 = 노인 기준 보행권 사각지대
- **청년 버퍼(초록 원)와 노인 버퍼의 차이** = 속도 격차로 인한 추가 소외 면적

### 차트에서 확인할 것

- `speed_gap_chart.png`: 두 막대 길이 차이가 클수록 속도 격차가 큰 자치구
- `welfare_green_scatter.png`: 왼쪽 하단(복지 낮음 + 녹지 낮음)에 위치한 자치구가 복합 취약 지역
- `dong_vulnerability.png`: 동 단위로 내려가면 자치구 내에서도 취약한 동이 드러남

### TOP 5 해석

`top5_urgent_districts.csv` / `dong_top5_report.csv`의 지역은 다음 조건을 동시에 만족합니다:
- 노인 기준 복지시설 보행권 커버리지가 낮음
- 공원 보행권 커버리지도 낮음
- 65세이상 인구 규모가 크거나, 박탈 노인 수가 절대적으로 많음
