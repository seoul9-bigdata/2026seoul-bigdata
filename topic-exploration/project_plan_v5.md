## 0. 프로젝트 정체성

### 0-1. 한 줄 정의

> **2040 서울도시기본계획이 약속한 "보행 30분 자족 생활권"은 표준 시민의 시간이다. 노인의 보행속도와 거동 특성을 반영하면 그 약속의 도달 영역은 크게 달라진다. 본 프로젝트는 "노인이 걸어서 닿는 서울"을 재측정하여, 보행일상권 재정비가 진행될 때 참고할 수 있는 진단 자료를 제공한다.**

### 0-2. 주제 초점 (v5에서 재정립)

**포함** — 노인이 **걸어서** 닿아야 하는 것들의 접근성:

- 생활 필수시설 (1차 의료·약국·시장·은행·주민센터·복지관)
- 위기 시각 대피시설 (무더위쉼터·한파쉼터)
- 사회적 접점 (공원·전통시장·종교시설·경로당)

**제외** — v5에서 기각한 것 (주제 부적합 사유):

- **응급의료 접근성 / 골든타임** — 응급차가 빨리 온다. 노인이 응급실을 찾아가는 관점이 아님. 주제 초점과 맞지 않음.
- **24시간 시설 운영시간 분석** — 폭염쉼터는 정오에 당연히 열린다. 응급실은 24시간. 운영시간 퍼센티지 지표는 허구의 문제의식.
- **주말 야간 외로움 시각 분석** — 근본적으로, 쉼터·복지관을 찾아다닐 수 있는 노인은 이미 찾아다닌다. 찾지 못하는 노인은 운영시간이 길어도 찾지 못한다. 시간축 자체의 문제의식이 빗나감.

### 0-3. 우리가 **하지 않는** 것 (명시적 기각 + 사유)

| 흔한 접근                                       | v5의 판단                                                                      |

## 1. 핵심 문제의식 (1페이지 발표용)

### 1-1. 2040 계획의 보이지 않는 가정

2040 서울도시기본계획은 "도보 30분 자족적 생활권"을 핵심 비전으로 제시한다(p.46). 그러나 이 30분은 **표준 시민의 보행속도(약 4 km/h, 1.1 m/s)** 를 가정한다. 같은 계획서는 시 인구의 19%(2025)가 65세 이상이며 2040년에는 32%에 이를 것이라고도 명시한다(p.27). 약속된 시간과 그 약속을 받을 시민의 시간이 일치하지 않는다.

### 1-2. 노인의 보행속도는 다르다 (출처 있는 수치)

| 차원                    | 표준 시민 | 65~74세  | 75~84세  | 85세+    |
| ----------------------- | --------- | -------- | -------- | -------- |
| 보행속도 (m/s)          | 1.1 ~ 1.4 | 0.95     | 0.78     | 0.58     |
| 도보 30분 도달 거리 (m) | 약 2,000  | 약 1,710 | 약 1,400 | 약 1,040 |
| 1 km 도달 시간 (분)     | 12.5      | 17.5     | 21.4     | 28.7     |

> 출처: Bohannon (1997); Studenski et al. (2011) JAMA. 한국 노인 평균 0.78 m/s 채택.

→ **계획의 30분 약속은 65~74세에게는 약 35분, 75세 이상에게는 약 45분, 85세 이상에게는 약 60분**.

### 1-3. 생활권 접근성의 7개 차원 (어휘 부활)

노인이 **걸어서** 닿아야 하는 동네 자원은 다층적이다:

| 차원          | 예시 시설                          | 노인에게 결정적인 이유      |
| ------------- | ---------------------------------- | --------------------------- |
| 의료·보건     | 1차 진료기관, 약국                 | 만성질환 관리, 일상 약 조제 |
| 교통·이동     | 저상버스 정류장, 지하철 엘리베이터 | 권역 밖 이동의 시작점       |
| 복지·커뮤니티 | 경로당, 노인복지관                 | 일상 돌봄, 사회적 관계      |
| 생활 인프라   | 전통시장, 슈퍼마켓, 은행           | 소량 구매 가능, 대면 금융   |
| 안전·치안     | 주민센터, CCTV 밀집                | 야간 보행 안심              |
| 기후재난      | 무더위쉼터, 한파쉼터               | 위기 시각 생명 보호         |
| 사회적 관계망 | 공원, 종교시설                     | 고립 방지, 일상 접점        |

본 프로젝트는 이 7개 차원을 **노인 보행속도 30분 기준**으로 재측정한다. 이것이 "노인이 걸어서 닿는 서울"의 실제 모습이다.

### 1-4. 공간 평등 ≠ 시간 평등

> **물리적 공간은 평등하지만, 생물학적 시간은 불평등하다. 정책의 표준이 누군가에게는 실질적 배제의 기준이 된다.**
>
> 본 진단은 2040 도시기본계획을 부정하지 않는다. 계획이 내세운 "보행일상권"이라는 방향성은 옳다. 다만 그 방향성이 **노인을 포함한 모든 시민**에게 실제로 도달하려면, 표준 보행속도 기준의 재검토가 필요하다. 본 프로젝트는 그 재검토의 기초 자료를 제공한다.

## 2. 핵심 메시지 3개 (발표 결론으로 이어질 메시지)

### M1. 약속된 30분, 실제 60분

청년 보행 30분권과 노인 보행 30분권의 면적 격차를 동 단위로 측정. 그 격차에 거주하는 노인 인원 동시 산출.

### M2. 7개 차원의 누적 격차

노인 생활권의 7개 차원(의료·교통·복지·인프라·안전·기후·관계망)을 노인 보행속도 30분 기준으로 통합 측정. 동네마다 "무엇이 얼마나 멀어지는가"의 지문(Fingerprint)을 시각화.

### M3. 시설 1개의 체감 변화

가장 격차가 큰 동에서 가상 시설 1개 추가 시 체감 개선 시뮬레이션 제공. 정책 결정의 참고 자료로 기능 (결정 권한은 정책담당자).

## 3. 출발점 방법론 (v5 신규 — 피드백 9 대응, 투명성 섹션)

> 보행 접근성 분석의 **출발점**을 어떻게 정의하는지가 결과의 신뢰도를 좌우한다. 본 섹션은 이를 명시적으로 투명 공개한다.

### 3-1. 잘못된 접근: 단일 중심점

```
방법: 행정동 기하학적 중앙점(centroid)에서 출발하여 등시선 1개 생성
문제: (1) 노인 거주 밀집 지역 vs 실제 중심 불일치, (2) 산지·하천 포함된 동은 centroid가 무의미,
     (3) 동 면적 크면 격차 평가가 왜곡됨
```

### 3-2. v5 채택 방법: 격자 기반 다중 출발점 + 인구 가중 집계

```
Step 1. 서울 전역에 500m × 500m 격자 생성 (약 1,500개 셀)
Step 2. 각 격자 셀에 65세+ 추정 인구 분배 (dasymetric 보간 — §3-3 참고)
Step 3. 각 격자 셀의 centroid를 개별 출발점으로 삼아 노인 보행 30분 등시선 계산
Step 4. 격자별로 "30분권 내 도달 가능 시설 수 / 시설 유형 다양성" 산출
Step 5. 격자 결과를 행정동 경계로 집계하되, 격자 내 노인 인구를 가중치로 사용
     → 즉, "노인이 많이 사는 격자"의 결과가 동 집계에 더 큰 영향
```

### 3-3. 격자 인구 분배 (Dasymetric 보간)

동별 전체 인구·65세+ 인구는 통계청 KOSIS에서 제공되지만 격자 단위는 아니다. 격자에 인구를 분배하려면 "동 내 어디에 사람이 더 많이 사는가"를 알아야 한다.

**1차 방법**: 건물 footprint (국가공간정보포털)를 활용한 dasymetric 보간 — 주거용 건물 연면적에 비례해 분배.
**대안 (시간 부족 시)**: 통계청 SGIS 100m 격자 총인구 + 동별 65세+ 비율을 격자에 균등 적용.

### 3-4. 격자 접근법의 한계 명시 (발표 시 언급)

- OSM 보행망의 좁은 골목 누락 가능성
- 건물 footprint가 노후되거나 재개발 지역의 경우 실제와 차이
- 산지·공원 등 비거주 영역의 격자 필터링 필요 (Step 2에서 처리)

## 4. 분석 프레임 — 5개 모듈 (v4 6개 → v5 5개, 구성 재편)

### TB1. 보행 시간 격차 — 청년 30분권 vs 노인 30분권 (v4 유지)

> **위상**: 프로젝트 1번 시각화. 모든 후속 분석의 설득 기반. 피드백 5 반영.

**측정 대상**:

- 서울 전역에서 청년 30분권 면적 vs 노인 30분권 면적 격차
- 그 격차 영역에 거주하는 65세+ 인구 인원

**데이터**:
| # | 데이터 | ID | 출처 |
|---|--------|-----|------|
| 1 | 행정동 경계 shapefile | — | 통계청 SGIS |
| 2 | OSM 보행 네트워크 (서울) | — | OpenStreetMap |
| 3 | 격자 인구 (500m, 65+ 및 75+) | — | §3-3 방법 |

**핵심 산출 로직**:

```python
import osmnx as ox
import networkx as nx

G = ox.graph_from_place('Seoul', network_type='walk')

SPEED_YOUNG = 1.20   # m/s
SPEED_SENIOR = 0.78  # m/s (75+ 기준, 보수적)

def isochrone_polygon(G, start_node, speed_mps, time_min):
    travel_time = nx.single_source_dijkstra_path_length(
        G, start_node, cutoff=time_min*60,
        weight=lambda u, v, d: d['length'] / speed_mps)
    nodes_geom = [Point(G.nodes[n]['x'], G.nodes[n]['y'])
                  for n in travel_time]
    return alpha_shape(nodes_geom, alpha=0.005)

# 격자 셀마다 청년/노인 isochrone 두 개 생성
for _, cell in grid.iterrows():
    start = ox.distance.nearest_nodes(G, cell.x, cell.y)
    iso_young = isochrone_polygon(G, start, SPEED_YOUNG, 30)
    iso_senior = isochrone_polygon(G, start, SPEED_SENIOR, 30)
    cell['iso_young_area_km2'] = iso_young.area / 1e6
    cell['iso_senior_area_km2'] = iso_senior.area / 1e6
    cell['area_loss_ratio'] = 1 - iso_senior.area / iso_young.area

# 동별 집계 (노인 인구 가중)
df_tb1 = grid.groupby('adm_dr_cd').apply(
    lambda g: pd.Series({
        'mean_loss_ratio': (g.area_loss_ratio * g.pop_65plus).sum() / g.pop_65plus.sum(),
        'senior_in_loss_area': estimate_senior_in_gap_area(g),
    })
).reset_index()
```

**핵심 파생 변수**:
| 변수 | 단위 | 의미 |
|------|------|------|
| `mean_loss_ratio` | 0~1 | 동 내 격자들의 노인 인구 가중 평균 면적 손실 비율 |
| `senior_in_loss_area` | 명 | 청년 30분권에는 포함되나 노인 30분권에는 제외된 영역에 거주하는 65세+ 추정 인원 |

**시각화 산출물**:

- **TB1-A**: 단일 동 줌인 — 청년 isochrone(연한 색) 위에 노인 isochrone(진한 색) 중첩. 격자 출발점 다중 표시 (점 여러 개).
- **TB1-B**: 서울 전체 코로플레스 — `mean_loss_ratio` 동별 분포.
- **TB1-C**: 사다리 차트 (slope chart) — 청년/노인 두 축, 동별 격차 선 (TOP 30 동).
- **TB1-D**: 인원 가중 막대 차트 — TOP 20 동의 `senior_in_loss_area`. "면적 격차 1위와 인원 격차 1위가 다르다" 폭로.

**구현 가능성**: ★★★★☆

### TB2. 노인 생활권 7개 차원 실제 도달 격차 (v4 TB2 응급 전면 교체 — 피드백 3, 6)

> **위상**: 메인 진단 축. 7개 차원을 노인 보행속도 30분으로 통합. 피드백 1에서 되살린 "7개 차원" 어휘 통합.

**측정 대상**:

- 각 격자 셀의 노인 보행 30분 반경 내에 **7개 차원별 시설이 몇 개 있는가**
- 동별로 차원별 접근성을 집계하여 **"동네 인프라 지문(Fingerprint)"** 산출

**7개 차원별 대상 시설**:
| 차원 | 대상 시설 | 데이터 |
|------|----------|--------|

for _, cell in grid.iterrows():
    start = ox.distance.nearest_nodes(G, cell.x, cell.y)
    iso_senior = isochrone_polygon(G, start, SPEED_SENIOR, 30)

    for dimension, facilities in DIMENSION_FACILITIES.items():
        count = count_facilities_in_polygon(iso_senior, facilities)
        types = count_facility_types_in_polygon(iso_senior, facilities)
        cell[f'{dimension}_count'] = count
        cell[f'{dimension}_types'] = types

# 동별 집계 (노인 인구 가중)
df_tb2 = grid.groupby('adm_dr_cd').agg({
    f'{dim}_count': lambda s: np.average(s, weights=g.pop_65plus)
    for dim in DIMENSIONS

# 7개 차원 각각을 0~100 스케일로 정규화 (동 비교용)
for dim in DIMENSIONS:

    cell['touch_count'] = sum(

### 시뮬레이션 대상 2종 (v4 4종에서 축소)

응급 관련은 폐기되었으므로 시뮬레이션 대상도 축소:

# 자치구별로 TOP 후보지 3개씩만 (5명 팀이 다룰 현실적 개수)

## 5. 합성 지표 — "노인 생활권 접근성 지문"

v4의 "z-score 합성 총격차 지수"는 유지하되, TB5 "처방" 어휘 전환과 맞춰 **"취약 순위"가 아닌 "격차 지문"** 으로 제시.

```python
df_total = (df_tb1
    .merge(df_tb2, on='adm_dr_cd')
    .merge(df_tb3, on='adm_dr_cd')
    .merge(df_tb4, on='adm_dr_cd'))

# 동별 "생활권 격차 지문" — 7차원 + 위기 + 관계망
df_total['dimension_vector'] = df_total[[
    f'{dim}_norm' for dim in DIMENSIONS
] + [
    'heat_risk_z',    # TB3에서 z-score화
    'loneliness_load_z',  # TB4에서 z-score화
]].values.tolist()

# 보행 격차 단독 지표 (발표 핵심 수치)
df_total['minutes_gap'] = df_total['mean_loss_ratio'] * 30  # 대략적 분 환산
df_total['senior_affected_total'] = (
    df_total['senior_in_loss_area'] +
    df_total['solo_senior_not_covered']
)
```

## 9. 정책 참고 시나리오 (3가지, v5 겸손한 어휘)

### 시나리오 1: 보행 격차가 큰 동 (TB1)

> "강북구 X동은 노인 30분권 면적이 청년 30분권 대비 48% 수준으로 서울 하위 10%. 격차 영역에는 65세+ 인구 N명이 거주. **이 격차가 어디서 발생하는가를 들여다보면** 동 내부의 가파른 경사와 일방향 도로망이 주요 원인으로 추정됨. **재정비 시 참고 자료**."

### 시나리오 2: 차원 편중 동 (TB2 지문)

> "동작구 Y동의 인프라 지문은 교통·의료 차원은 서울 평균 상위이나 복지·관계망은 하위. 노인이 걸어서 닿을 수 있는 경로당·공원·종교시설이 **서울 평균의 1/3 수준**. 동의 65세+ 인구가 Z명인 점을 고려하면 **복지·관계망 차원의 재정비가 체감 개선 여지가 큰 영역**."

### 시나리오 3: 폭염 사각 독거노인 (TB3 + TB5)

> "관악구 W동: 폭염일수 자치구 평균 상위 10% + 75+ 독거노인 비율 30% + 노인 보행 15분 내 무더위쉼터 도달 가능 면적 12%. 사각 거주 독거노인 N명. **TB5 시뮬레이션 결과, 좌표 (위도·경도) 인근에 가상 쉼터 1개 추가 시 사각 독거노인이 N명 → M명으로 변화**. 단 최종 입지 결정은 인접 시설·토지 여건·지역 수요에 관한 추가 검토가 필요함."
