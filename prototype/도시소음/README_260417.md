## 📊 프로젝트 데이터 활용 및 분석 결과 요약

### 1. 활용 데이터 목록
분석에는 서울시에서 제공하는 공공데이터와 도시 데이터 센서(S-DoT) 데이터를 결합하여 활용했습니다.

| 데이터셋 명칭 | 형식 | 주요 활용 내용 |
| :--- | :---: | :--- |
| **서울시 역사마스터 정보** | CSV | 서울시 지하철 및 철도 역사의 명칭, 호선, 위치(위경도) 정보 추출 |
| **S-DoT 환경정보 설치 위치정보** | XLSX | S-DoT 센서의 모델 시리얼 번호와 설치 위치(위경도) 매핑 |
| **S-DoT NATURE (환경정보)** | CSV | 특정 기간(2026.03.30~04.05) 동안의 소음(평균/최대) 및 진동(X, Y, Z) 측정 데이터 |

---

### 2. 분석 및 데이터 처리 프로세스

1.  **공간 데이터 전처리 및 매핑**: 
    *   `BallTree` 알고리즘(Haversine 거리 기준)을 사용하여 각 철도 역사로부터 **반경 500m 이내**에 위치한 S-DoT 센서들을 식별했습니다.
2.  **대용량 데이터 필터링**: 
    *   수백만 행에 달하는 S-DoT 환경 측정 데이터를 `chunksize` 옵션으로 분할 로드하여, 역사 인근 센서(약 533개)의 데이터만 효율적으로 추출했습니다.
3.  **데이터 집계 및 시간대별 분석**: 
    *   24시간 주기의 패턴을 파악하기 위해 측정 시간을 시간대(Hour)별로 그룹화하고, 소음과 진동 수치의 평균 및 최댓값을 산출했습니다.
4.  **역사별 환경 지표 산출**: 
    *   역사 영향권 내의 모든 센서 데이터를 평균하여, 각 역사가 노출된 전반적인 소음 및 진동 환경 지표를 생성했습니다.
