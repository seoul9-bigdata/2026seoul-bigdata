# D3. 의료 — 청년·노인 30분 의료 접근성 분석

**담당: 이정태**  
**핵심 질문: 노인이 만성질환 관리를 동네에서 할 수 있는가?**

---

## 분석 개요

서울시 의원·병원·보건소·약국을 대상으로,  
청년(1.2 m/s)과 노인(0.78 m/s) 보행속도 기준 30분 도달 가능 시설 수를 동 단위로 비교.

| 구분 | 보행속도 | 30분 도달 거리 |
|------|---------|--------------|
| 청년 | 1.20 m/s | **2,160 m** |
| 노인 | 0.78 m/s | **1,404 m** |

**서울 평균 손실률: 55.9%** — 노인은 청년보다 평균 56% 적은 의료시설에 30분 내 접근 가능.
