# D3. 의료 — 청년·노인 30분 의료 접근성 분석

**담당: 이정태**  
**핵심 질문: 노인이 만성질환 관리를 동네에서 할 수 있는가?**

---

## 분석 개요

서울시 의원·병원·보건소·약국을 대상으로,  
청년(1.28 m/s)과 보행보조 노인(0.88 m/s) 보행속도 기준 30분 도달 가능 시설 수를 동 단위로 비교.  
노인 반경은 **Tobler's Hiking Function**으로 동별 경사도를 반영해 추가 보정.

| 구분 | 보행속도 | 30분 도달 거리 (평지) |
|------|---------|---------------------|
| 청년 | 1.28 m/s | **2,304 m** |
| 보행보조 노인 | 0.88 m/s | **1,584 m** |
| 경사 보정 노인 (서울 평균) | ~0.70 m/s | **~1,268 m** |

> **속도 출처**: 한음 외(2020) — 팀 공통 파라미터 표준  
> **거리 측정**: 동 centroid 기준 유클리드 + Tobler 경사 보정 (EPSG:5179)

## 실행 방법

```bash
# 가상환경 활성화
source venv/Scripts/activate   # Windows bash

# 패키지 설치 (최초 1회)
pip install -r requirements.txt

# 분석 실행 (순서대로)
python src/01_medical_access.py        # 기본 접근성 분석 (청년 vs 노인, 경사 보정)
python src/02_temporal_access.py       # 시간대별 접근성 (평일/저녁/주말)
python src/03_trip_chaining.py         # 삼각 동선 (병원→약국→집)
python src/04_slope_correction.py      # 경사도 보행속도 보정 (Tobler)
python src/05_elderly_population.py    # 고령 인구 이중 취약지 분석
```

## 방법론 한계 및 향후 개선

- 거리 측정: 동 centroid 유클리드 + Tobler 경사 보정. 실제 OSM 보행 네트워크(다익스트라) 적용 시 손실률 추가 상승 예상
- 이론적 손실률 `1 - (0.88/1.28)²` ≈ 52.7% vs 실측 51.7% → 경사를 동등 적용하면 이론값에 근접. 나머지 오차는 시설 공간 분포 불균형 반영
- 동별 고령 인구: 양석준 공유 고령자현황 CSV (2026년 4월 기준)
