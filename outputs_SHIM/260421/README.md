# 🚀 서울시 노인 보행일상권 시각화 : ④ 생활 인프라

> **"도시의 설계 속도는 누구를 기준으로 하는가?"** > 본 프로젝트는 보행 약자(노인)의 실제 보행 속도를 반영하여, 서울시 내 필수 생활 인프라(전통시장, 슈퍼마켓 등)의 실질적 접근성을 시각화하고 정책적 사각지대를 조명합니다.

---

## 1. 프로젝트 배경 및 핵심 로직
단순한 물리적 거리(Radius)가 아닌, **보행 특성별 이동 시간**을 기준으로 일상권의 범위를 재정의했습니다. 특히 법적 기준보다 느린 '현실의 속도'에 집중했습니다.

* **일반인 (평균):** `1.28 m/s` — 표준 도시 설계의 기준
* **고령자 (평균):** `1.12 m/s` — 노화에 따른 자연스러운 감속 반영
* **보행보조기 사용 노인 (하위 15%):** **`0.70 m/s`**
    * **Insight:** 현재 교통약자 보호구역 기준(`0.8 m/s`)보다 낮은 수치로, 기존 법적 보호망이 미처 고려하지 못한 '최약계층'의 이동권 문제를 데이터로 증명합니다.

---

## 3. 주요 시각화 기능 (Features)

### 🛰️ 인프라 별자리 지도 (Infrastructure Constellation)
* **개념:** 행정동 중심점으로부터 각 인프라 지점까지의 거리를 별자리의 형태로 시각화했습니다.
* **인터랙티브:** 보행 유형을 선택함에 따라 동일한 시간 내에 도달 가능한 '별'의 개수가 급격히 줄어드는 과정을 통해 **일상권의 수축**을 직관적으로 보여줍니다.

### 📊 서울시 보행권 지수 대시보드
* **등급 시스템:** 서울시 25개 자치구별로 0.70 m/s 기준 도달 가능성을 계산하여 **양호 / 보통 / 미흡** 등급을 부여합니다.
* **보행 시뮬레이터:** 횡단보도 녹색 신호 시간 내에 실제 노인(0.70 m/s)이 도로를 끝까지 건널 수 있는지 여부를 실시간으로 시뮬레이션하여 보여줍니다.

---

## 4. 분석 결과 요약
* **보행 격차 확인:** 보행보조기를 사용하는 노인의 경우, 일반인 대비 동일 시간 내 접근 가능한 인프라 수가 **최대 60% 이상 감소**하는 것으로 나타났습니다.
* **인프라 양극화:** 특정 구(구로구, 강동구 등)의 실데이터 POI 분석 결과, 전통시장과 슈퍼마켓의 배치가 보행 약자의 보행 반경을 벗어나는 사각지대가 다수 발견되었습니다.
