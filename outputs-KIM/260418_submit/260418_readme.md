# 260418 작업 요약 — 서울 노인 보행권 분석

**작성자**: 김성령  
**작업일**: 2026-04-18  
**핵심 질문**: _"2040 서울도시기본계획의 도보 30분 자족 생활권은 누구의 30분인가?"_

---

## 배경 및 목적

서울시는 2040 도시기본계획에서 '도보 30분 자족 생활권'을 핵심 비전으로 제시한다.  
그러나 이 "30분"은 건강한 성인을 기준으로 설계된 것이다.  
보행보조장치(워커·지팡이)를 사용하는 노인은 평균 보행속도가 일반인의 약 69% 수준(0.88 m/s vs 1.28 m/s)에 그친다.  
이 분석은 **동일 시간(30분) 안에 발생하는 보행 도달권 격차**를 서울 전체 행정동 단위로 정량화하고 시각화하는 것을 목표로 한다.

> 보행속도 출처: 한음 외 (2020), 한국ITS학회논문지 19(4), n=4,857

## 방법론

### 핵심 지표: 보행 네트워크 커버리지 손실률

```
손실률(%) = (1 - 보조장치 도달 노드 수 / 일반인 도달 노드 수) × 100
```

- **일반인**: 30분 × 1.28 m/s = 최대 2,304m 이동 가능
- **보조장치 노인**: 30분 × 0.88 m/s = 최대 1,584m 이동 가능

**노드(node)의 정의**: OSM 보행 그래프의 노드는 도로의 **교차점·분기점**이다. 병원·마트·공원 같은 목적지(POI)가 아니다. 즉 손실률이 측정하는 것은 "어떤 특정 목적지에 도달할 수 있는가"가 아니라, **"30분 안에 보행 네트워크의 어느 지점까지 이동할 수 있는가"** — 보행 네트워크 커버리지의 차이다.

**출발점의 정의**: 각 동의 행정구역 기하학적 중심점(centroid)에서 가장 가까운 노드 **1개**를 출발점으로 사용한다. 동 안에 거주하는 실제 주민들의 분산된 출발지를 대표하지 않으며, 이는 분석의 주요 한계다.

- 노드 수를 도달 가능 면적의 프록시로 사용 (그래프 밀도 균일 가정)
- 이상적인 격자 도시에서 이론적 손실률 ≈ `(0.88/1.28)² × 100 ≈ 53%`이며, 서울 실측 평균 54.7%와 일치

### 등시선(Isochrone) 계산

- 알고리즘: 단일 출발점 다익스트라 (Dijkstra) — `nx.single_source_dijkstra_path_length`
- 가중치: `엣지 길이(m) / 보행속도(m/s)` = 통과 시간(초)
- 유향 그래프를 무향 그래프로 변환하여 방향 편향 제거
- 도달 노드 집합에 concave hull을 적용해 폴리곤 생성

### 서울 전체 병렬 계산

- 426개 행정동 × 2회(일반인/보조장치) = 852회 Dijkstra 연산
- Python `multiprocessing.Pool` (6 workers) 사용 — 소요 약 8분
- 결과를 CSV 캐시로 저장하여 재실행 시 생략

## 주요 발견

### 1. 손실률: 서울 평균 54.7%

손실률의 정확한 의미:

> "동의 중심 부근 교차점 하나에서 출발할 때, 30분 안에 일반인이 도달할 수 있는 보행 네트워크 교차점·분기점 중 약 55%에 보행보조장치 노인은 시간 내 도달하지 못한다"

직관적으로: 같은 30분 동안 일반인은 약 2.3km 반경의 도로망을 커버하지만, 보조장치 노인은 약 1.6km까지만 커버한다. 그 면적 차이가 손실률이다.  
이론값(53%)과 거의 일치하며, 일부 지역은 80~90%까지 올라간다.

**주의**: 이 지표는 POI(병원·마트 등) 접근성 분석이 아니다. 보행 네트워크 커버리지 차이를 정량화한 것이며, 실제 목적지 도달 여부를 직접 측정하지는 않는다.

### 2. 손실률의 결정 요인

| 요인                                  | 상관계수(r)          | 해석                                           |
| ------------------------------------- | -------------------- | ---------------------------------------------- |
| 네트워크 접근성 (일반인 도달 노드 수) | **-0.254** (p<0.001) | 도로 연결성이 좋을수록 격차 완화               |
| 고령화율                              | 0.020 (p=0.68)       | 무상관 — 노인이 많이 산다고 손실률이 높지 않음 |
| 노인 절대 인구                        | -0.105 (p=0.03)      | 미약한 음의 상관                               |

→ **손실률은 인구 구성이 아니라 도시 형태(도로 밀도·연결성)가 결정한다.**

### 3. 정책 우선순위: "영향받는 노인 수"

손실률 단독으로는 정책 우선순위를 정할 수 없다.  
손실률이 높아도 노인이 적은 동은 개선 효과가 작고, 손실률이 중간이어도 노인이 많은 동은 더 많은 사람이 혜택을 받는다.

```
영향받는 노인 수 = 손실률(%) × 65세+ 인구 / 100
```

| 순위 | 행정동         | 손실률 | 65세+ 인구 | 영향받는 노인 |
| ---- | -------------- | ------ | ---------- | ------------- |
| 1    | 노원구 상계1동 | 79.8%  | 9,013명    | **7,191명**   |
| 2    | 양천구 신정3동 | 60.5%  | 9,466명    | 5,722명       |
| 3    | 은평구 불광1동 | 66.7%  | 8,360명    | 5,574명       |
| 4    | 은평구 진관동  | 53.6%  | 10,203명   | 5,467명       |
| 5    | 중랑구 신내1동 | 63.5%  | 8,514명    | 5,407명       |

→ **서울 전체 약 100만 명**의 65세+ 노인이 보행 손실권 내에 거주한다.

## 한계 및 유의사항

- **출발점 단일화**: 각 동에서 centroid 부근의 교차점 1개만 출발점으로 사용. 동 내 주민들의 실제 출발지(가정, 경로당 등)를 대표하지 않음. 출발점 위치에 따라 손실률이 달라질 수 있음
- **POI 접근성 미측정**: 노드는 도로 교차점이며 병원·마트·복지관 등 목적지가 아님. "어떤 시설에 갈 수 있는가"는 별도 분석 필요
- **경사도 미반영**: 현재 모델은 모든 도로를 평지로 가정. 실제 노인 이동에서 경사는 결정적 변수
- **노드 수 = 면적 프록시**: 그래프 밀도가 동 전체에서 균일하다는 가정 하에 사용. 실제 concave hull 면적 계산보다 오차 있음
- **보행속도 단일 값**: 한 연구의 집단 평균값 사용. 개인차·날씨·도로 노면 조건 미반영
