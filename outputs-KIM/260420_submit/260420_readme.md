## Part A — 보행 도달권 시각화 (260418 개선)

### A1: 5개 대표 지점 × 3속도 그룹 × 시간 슬라이더

#### 개선 배경

260418 버전은 강동구 길동사거리 **1개 고정 지점** 에서만 등시선을 보여줬다. 팀 발표 맥락에서 서울 다양한 지역의 격차를 비교하려면, 여러 출발점과 속도 그룹을 **동시에** 탐색할 수 있어야 한다.

#### 보행속도 기준

보행속도 출처: 한음 외 (2020), 한국ITS학회논문지 19(4), n=4,857

| 그룹 | 속도 | 30분 도달 거리 |
| ---- | ---- | -------------- |
| 일반인 (65세 미만) | **1.28 m/s** | 2,304 m |
| 65세 이상 노인 | **1.12 m/s** | 2,016 m |
| 보행보조장치 사용 노인 | **0.88 m/s** | 1,584 m |

#### 5개 출발 지점

| # | 지점 | 선정 이유 |
| - | ---- | --------- |
| P0 | 종로 탑골공원 | 서울 원도심, 고령 인구 밀집, 격자형 도로망 |
| P1 | 강북구 미아역 | 북부 고령화 지역, 구릉지 특성 |
| P2 | 관악구 신림동 | 남부 경사지, 청년·노인 혼재 주거 |
| P3 | 강남구 삼성역 | 도로 연결성 최우수 지역과의 대비 |
| P4 | 송파구 잠실새내 | 한강변 + 대규모 주거단지 |

#### 주요 발견

- 45분 기준, 일반인 도달 면적 대비 보행보조장치 노인의 도달 면적은 평균 **약 55% 수준**
- 강남(삼성역) 일반인 45분: **26.5 km²** vs 보조장치 노인: **12.0 km²** — 동일 시간에 절반도 안 됨
- 잠실새내(한강변): 일반인 **21.6 km²** vs 보조장치 노인 **9.6 km²** — 강·고속도로 단절 영향 뚜렷
- 랜드마크 도달 분류: 초록(전 그룹 도달) / 노랑(일반인만 도달) / 회색(모두 불가)

#### 알고리즘

```
등시선 계산:
  가중치 = 엣지 길이(m) / 보행속도(m/s)  →  통과 시간(초)
  nx.single_source_dijkstra_path_length(G, start, cutoff=T_sec)
  도달 노드 집합 → concave_hull(ratio=0.05) → 폴리곤

캐시: 5점 × 3그룹 × 3시간 = 45개 등시선 + 랜드마크 도달 여부 → JSON 저장
```

## Part B — 기후 도메인: 폭염·한파·결빙

### 공통 방법론

모든 B 시각화에서 접근성 분석의 기준은 동일하다.

```
일반인: 1.28 m/s
보행보조장치 노인: 0.88 m/s

15분 도달 거리:  일반인 1,152m  /  노인 792m
30분 도달 거리:  일반인 2,304m  /  노인 1,584m
```

행정동(426개) 중심점 → 그래프 노드 스냅 → **멀티소스 Dijkstra** (모든 쉼터 노드 동시 출발) → 동별 최단 거리 산출 → 접근성 분류

```
분류 기준:
  both_15  : 거리 ≤ 792m  → 둘 다 15분 이내
  young_15 : 792 < 거리 ≤ 1152m  → 일반인만 15분
  both_30  : 1152 < 거리 ≤ 1584m → 둘 다 30분 이내
  young_30 : 1584 < 거리 ≤ 2304m → 일반인만 30분
  neither  : 거리 > 2304m  → 둘 다 30분 초과
```

### B1: 폭염쉼터 노인 보행 접근성

**데이터**: 서울시 무더위쉼터 4,107개

| 분류 | 동 수 | 비율 |
| ---- | ----- | ---- |
| 둘 다 15분 이내 | 398 | 93.4% |
| 일반인만 15분 | 13 | 3.1% |
| 둘 다 30분 이내 | 12 | 2.8% |
| 일반인만 30분 | 3 | 0.7% |
| 둘 다 30분 초과 | 0 | 0.0% |

**해석**: 서울의 폭염쉼터는 높은 밀도(km²당 6.8개)로 분포하여, 93%의 행정동에서 보행보조장치 노인도 15분 이내에 도달 가능하다. 단, 서울 외곽 및 산지 인접 동에서 접근 격차가 발생한다.

| 둘 다 30분 이내 | 13 | 3.1% |

> **"더운 여름보다 추운 겨울에, 서울 노인의 쉼터 접근성이 더 나쁘다."**

### B3: 겨울 결빙 취약지점 — 제설함 커버리지 + 도로열선

#### 분석 개요

겨울 결빙은 노인 보행의 가장 직접적인 물리적 위험이다. 제설함이 없으면 제설 자체가 불가능하고, 열선이 없으면 상습 경사 결빙 구간이 방치된다.

```
제설함이 없는 곳 = 제설 미시행 가능성 ↑
               = 결빙 지속 ↑
               = 낙상 위험 ↑
               = 노인 보행 단절
```

#### 제설함 커버리지 분석

**데이터**: 서울시 제설함 위치정보 10,437개 (EPSG:5186 좌표)

| 커버 반경 | 커버 면적 | 서울 전체 대비 |
| --------- | --------- | -------------- |
| 100m | 163.1 km² | **26.9%** |
| 200m | 328.0 km² | **54.1%** |

**해석**: 제설함 200m 반경으로 커버되는 면적이 전체 서울의 54%에 불과하다. 나머지 **45.9%** 는 제설 미도달 가능 구역 = 결빙 취약지대. 이 지역은 노인 보행에 직접적 위협이다.

#### 도로열선 현황

**데이터**: 도로열선 설치 현황 (25개 구, 총 연장 집계)

- 서울시 열선 설치 = 상습 결빙·급경사 구간 공식 인정
- 구별 총 연장 버블 지도로 시각화 (버블 크기 ∝ 열선 총 연장)
- 열선이 많은 구 = 공식적으로 결빙 위험이 높다고 인정된 곳

**두 레이어 조합으로 읽는 법**:
- 열선은 있는데 제설함 미커버 → 위험을 알지만 사후 대응만 있는 구간
- 열선도 없고 제설함도 미커버 → 행정 사각지대

## 주요 발견 종합

### 1. Part A — 보행 도달권 격차는 지역보다 도로망이 결정한다

260418에서 도출한 핵심 명제("손실률은 인구 구성이 아니라 도시 형태가 결정한다")를 5개 지점 비교로 시각적으로 재확인.  
강남(삼성역) 일반인의 45분 도달 면적(26.5 km²)이 가장 넓고, 잠실새내(한강변 단절)가 21.6 km²로 그 다음. 강북·관악은 구릉지 영향으로 더 좁다.  
**어느 지점에서나 보조장치 노인은 일반인 면적의 45~48%만 커버한다.**

### 2. Part B — 기후 인프라 접근성에도 동일한 격차가 존재한다

| 시설 | 개수 | 노인 15분 OK 동 비율 | 격차 비율 |
| ---- | ---- | ------------------- | --------- |
| 폭염쉼터 | 4,107개 | 93.4% | 6.6% |
| 한파쉼터 | 1,642개 | 89.2% | **10.8%** |

한파쉼터가 폭염쉼터보다 사각지대가 크다 — 추운 계절에 오히려 격차가 더 벌어진다.

### 3. Part B3 — 서울 면적의 46%는 제설함 도달 범위 밖

100m 커버리지 기준으로 서울의 73%가 미커버 상태. 200m로 넓혀도 46%가 제설함 범위 밖. 열선 데이터와 겹쳐보면, **공식적으로 결빙 위험이 인정된 구간인데도 제설 인프라가 부족한 지역**이 드러난다.

## 한계 및 유의사항

- **행정동 중심점 단일 출발**: 동 내 실제 주거 분포를 반영하지 않음. 출발점 위치에 따라 분류 결과 달라질 수 있음
- **멀티소스 Dijkstra의 cutoff 적용**: 연산 효율을 위해 최대 탐색 거리를 일반인 30분 도달거리(2,304m)의 1.05배로 제한. 그 이상 거리는 "둘 다 불가"로 일괄 처리됨
- **제설함 좌표계**: 원본 데이터 좌표가 EPSG:5186 mm 단위. 역삼2동 샘플로 좌표 검증 후 변환 적용
- **열선 데이터 미지오코딩**: 열선 CSV는 도로 구간 텍스트만 있어 정확한 좌표 매핑 불가. 구 단위 총 연장 집계로 대체
- **결빙 위험 = 제설함 미커버 전제**: 제설함이 없어도 구청 차량 제설이 이루어질 수 있음. 본 분석은 '사전 준비 인프라'의 공간적 분포를 측정하며, 실제 결빙 발생 여부와 직접 대응되지 않음
